##生成第二的hw1-1 新增了直接輸入數值的功能


請生成一個 Streamlit 應用程式，用來演示簡單線性回歸模型。應用程式應該允許用戶調整參數 `a`, `b`, `c` 以及噪聲的方差，並實時顯示生成的數據點和回歸線。調整參數的方式應包括滑動條和數字輸入框，這些控制元件應該位於側邊欄。

生成的隨機數據點應該滿足以下條件：
- `x` 的範圍從 -10 到 10，均勻分布
- `y = a*x + b + c*noise`，其中 `noise` 是服從平均值為 0 的正態分布的噪聲，方差由用戶設定

應用程式的需求：
1. 允許用戶調整參數 `a`（範圍從 -10 到 10），`b`（範圍從 0 到 100），`c`（範圍從 0 到 100），和噪聲的方差（範圍從 0 到 10）
2. 允許用戶調整數據點的個數 `N`（範圍從 10 到 1000）
3. 實時顯示生成的數據點和回歸線
4. 使用 scikit-learn 庫進行線性回歸建模
5. 顯示模型參數（估計的斜率和截距）

生成的程式碼應該包含如下部分：
- 一個用來生成數據的函數
- Streamlit UI 用來調整參數，包括滑動條和數字輸入框
- Matplotlib 繪製數據點和回歸線
- 在頁面上顯示模型參數（估計的斜率和截距）
